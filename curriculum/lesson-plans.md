# Detailed Lesson Plans - Zordic Music Studio Curriculum

## Unit 1: Fundamentals of Sound and Music (Weeks 1-2)

### Lesson 1: Introduction to Sound Waves

**Duration:** 90 minutes
**Difficulty:** Beginner

#### Lesson Flow

**Opening (10 minutes)**
- Ice breaker: "What's your favorite song and why?"
- Introduction to Zordic Music Studio interface
- Overview of the journey ahead

**Direct Instruction (20 minutes)**
- Theory-agent presents: Physics of sound waves
- Visual demonstrations using Zordic's oscillator and spectrum analyzer
- Key concepts: frequency (Hz), amplitude (dB), wavelength
- Digital audio basics: sampling rate, bit depth

**Guided Practice (30 minutes)**
- Exercise 1: Wave Explorer (15 min)
  - Students manipulate sine wave frequency from 20Hz to 20kHz
  - Observe visual changes in waveform display
  - Listen to how pitch changes with frequency
  - Agent provides real-time feedback

- Exercise 2: Frequency Hunt (20 min)
  - Play everyday sounds (doorbell, car horn, voice)
  - Students use spectrum analyzer to identify frequencies
  - Agent guides interpretation of spectrum display

**Independent Practice (20 minutes)**
- Exercise 3: Amplitude Control
  - Students create volume fades and swells
  - Experiment with decibel levels
  - Learn about dynamic range

**Closure & Assessment (10 minutes)**
- Quick quiz via theory-agent (adaptive difficulty)
- Preview creative challenge
- Exit ticket: "Describe frequency in your own words"

#### Creative Challenge: Create Your Sonic Signature

**Assignment Overview:**
Design a unique 5-second sound using only basic waveforms that represents your personality.

**Instructions:**
1. Open new Zordic project
2. Choose 3+ different waveforms (sine, square, saw, triangle)
3. Adjust frequencies to create interesting combinations
4. Use amplitude changes to add movement
5. Export as "FirstName_LastName_SonicSignature.wav"

**Assessment Criteria:**
- Technical (50%): Uses 3+ waveforms, demonstrates frequency/amplitude control, proper export
- Creative (30%): Unique sound, personal expression, experimentation
- Documentation (20%): Proper file naming, includes written description

**Agent Support:**
- Students can ask theory-agent for waveform recommendations
- Demo-agent provides real-time feedback on technical execution

### Lesson 2: Rhythm, Beats, and Time

**Duration:** 90 minutes
**Difficulty:** Beginner

#### Lesson Flow

**Opening (10 minutes)**
- Play 3 songs with different BPMs (slow ballad, mid-tempo pop, fast EDM)
- Students guess the tempo
- Discuss how tempo affects mood

**Direct Instruction (20 minutes)**
- Rhythm-agent explains:
  - BPM (beats per minute) concept
  - Common time signatures (4/4, 3/4, 6/8)
  - Musical note values (whole, half, quarter, eighth, sixteenth)
- Live demonstration on Zordic's drum grid

**Guided Practice (35 minutes)**
- Exercise 1: BPM Explorer (15 min)
  - Agent sets metronome at various tempos (60, 90, 120, 140, 180 BPM)
  - Students tap along and identify genre associations
  - Practice setting BPM in Zordic

- Exercise 2: Time Signature Detective (20 min)
  - Listen to song excerpts
  - Count beats and identify time signatures
  - Agent provides hints and feedback
  - Discussion of how time signatures create feel

**Independent Practice (15 minutes)**
- Exercise 3: Grid Master
  - Students program their first beat on step sequencer
  - Start with simple kick-snare pattern in 4/4
  - Experiment with adding hi-hats
  - Use quantization to align timing

**Closure & Assessment (10 minutes)**
- Student volunteers share their beat
- Class votes on "grooviest" beat
- Rhythm-agent provides technique tips

#### Creative Challenge: Rhythm Around the World

**Assignment Overview:**
Create a 16-bar rhythmic pattern inspired by a world music genre (Afrobeat, Samba, Reggaeton, Bollywood, etc.).

**Instructions:**
1. Research your chosen genre (listen to 3+ examples)
2. Identify characteristic rhythm patterns
3. Set appropriate BPM in Zordic
4. Program 16-bar pattern on drum grid
5. Use at least 4 different percussion sounds
6. Include A and B sections for variation
7. Export audio and document your research

**Assessment Criteria:**
- Research (20%): Demonstrates knowledge of genre
- Technical (40%): Accurate rhythm programming, appropriate BPM, 4+ sounds
- Musicality (30%): Grooves well, shows variation, genre-appropriate
- Documentation (10%): Written explanation of genre choice and techniques

### Lesson 3: Music Theory Fundamentals

**Duration:** 90 minutes
**Difficulty:** Beginner

#### Lesson Flow

**Opening (10 minutes)**
- Play major scale - ask students to describe feeling (happy, bright)
- Play minor scale - ask students to describe feeling (sad, dark)
- Introduction: "Music theory is a language to express emotions"

**Direct Instruction (25 minutes)**
- Theory-agent presents:
  - Musical alphabet (A-G)
  - Piano keyboard layout (white keys = natural notes, black keys = sharps/flats)
  - Staff notation basics
  - Intervals (distance between notes)
  - Major scale formula (W-W-H-W-W-W-H)
  - Minor scale formula (W-H-W-W-H-W-W)

**Guided Practice (35 minutes)**
- Exercise 1: Note Navigator (20 min)
  - Interactive keyboard on screen
  - Agent calls out notes, students find them
  - Students play notes, agent identifies them
  - Learn octave concept (same note, different pitch)

- Exercise 2: Interval Ear Training (20 min)
  - Agent plays intervals (minor 2nd through octave)
  - Students identify by ear
  - Associate intervals with emotions and famous songs
  - Practice in Zordic piano roll

**Independent Practice (15 minutes)**
- Exercise 3: Scale Builder
  - Students construct C major scale using formula
  - Build A minor scale
  - Experiment with playing melodies using only scale notes

**Closure & Assessment (5 minutes)**
- Quick keyboard quiz
- Preview emotional journey challenge

#### Creative Challenge: Emotional Journey

**Assignment Overview:**
Create two 8-bar melodies: one happy (major) and one sad (minor).

**Instructions:**
1. Happy melody:
   - Use C major scale (C-D-E-F-G-A-B-C)
   - 8 bars in Zordic piano roll
   - Range of at least one octave
   - Upward melodic contour preferred

2. Sad melody:
   - Use A minor scale (A-B-C-D-E-F-G-A)
   - 8 bars in Zordic piano roll
   - Range of at least one octave
   - Downward or static melodic contour

3. Play both for a peer - can they identify which is happy vs sad?

**Assessment Criteria:**
- Theory (40%): Correct scale usage, notes within scale, octave range
- Expression (40%): Melody conveys intended emotion, musical phrasing
- Technical (20%): Proper piano roll usage, clean export

### Lesson 4: Tones and Timbre

**Duration:** 90 minutes
**Difficulty:** Beginner

#### Lesson Flow

**Opening (10 minutes)**
- Play same note on piano, guitar, trumpet, synth
- Question: "Why do they sound different?"
- Introduction to timbre (tone color)

**Direct Instruction (20 minutes)**
- Synth-agent explains:
  - Timbre = unique sonic fingerprint
  - Harmonics and overtones
  - Basic waveforms and their characteristics:
    - Sine: Pure, smooth, fundamental only
    - Square: Hollow, clarinet-like, odd harmonics
    - Sawtooth: Bright, brassy, all harmonics
    - Triangle: Mellow, flute-like, diminishing odd harmonics
  - ADSR envelope introduction

**Guided Practice (40 minutes)**
- Exercise 1: Timbre Detective (15 min)
  - Agent plays instrument sounds
  - Students identify by timbre alone
  - Discuss what makes each unique
  - View spectrum analysis

- Exercise 2: Waveform Workshop (20 min)
  - Students load each basic waveform in Zordic synth
  - Play same note (A440) with each
  - Observe visual differences in waveform
  - Listen to harmonic differences
  - Experiment with mixing waveforms

**Independent Practice (15 minutes)**
- Exercise 3: Sound Designer
  - Create 5 different timbres using basic synthesis:
    1. Deep bass
    2. Bright lead
    3. Soft pad
    4. Plucky sound
    5. Experimental/weird
  - Use oscillator, basic filter, envelope

**Closure & Assessment (5 minutes)**
- Students share most interesting sound
- Synth-agent provides tips for improvement

#### Creative Challenge: Synth Sound Library

**Assignment Overview:**
Design a collection of 8 distinct synth sounds for different musical purposes.

**Instructions:**
1. Create 2 of each category:
   - Bass sounds (sub bass, mid bass)
   - Lead sounds (aggressive, melodic)
   - Pad sounds (warm, bright)
   - Pluck sounds (short, percussive)

2. For each sound:
   - Choose appropriate waveform(s)
   - Shape with envelope (ADSR)
   - Apply basic filtering if needed
   - Name descriptively
   - Save as Zordic preset

3. Create a document listing:
   - Sound name
   - Intended use
   - Waveform used
   - Key settings (envelope, filter)

**Assessment Criteria:**
- Variety (30%): Distinct timbres across categories
- Technical (40%): Appropriate waveform choices, functional sounds
- Organization (20%): Clear naming, documentation
- Creativity (10%): Unique or experimental sounds included

## Unit 2: Beat Making and Production (Weeks 3-4)

### Lesson 5: Drum Patterns and Percussion

**Duration:** 90 minutes
**Difficulty:** Beginner-Intermediate

#### Lesson Flow

**Opening (10 minutes)**
- Listen to famous drum intros (Phil Collins "In the Air Tonight", Led Zeppelin "When the Levee Breaks")
- Discuss what makes drums exciting
- Introduction to drum kit components

**Direct Instruction (20 minutes)**
- Beat-agent demonstrates:
  - Basic drum kit: kick, snare, hi-hat, toms, crash, ride
  - Role of each drum:
    - Kick = foundation, pulse
    - Snare = backbeat, drive
    - Hi-hat = rhythm, groove
  - Basic rock pattern
  - Basic hip-hop pattern
  - Basic EDM pattern
  - Velocity and dynamics
  - Swing and humanization

**Guided Practice (40 minutes)**
- Exercise 1: Drum Kit Explorer (15 min)
  - Students trigger each drum element
  - Experiment with different kick/snare/hat sounds
  - Learn difference between acoustic and electronic drums

- Exercise 2: Pattern Library (30 min)
  - Beat-agent provides genre templates
  - Students program classic patterns:
    - 4/4 rock (kick 1&3, snare 2&4, 8th hi-hats)
    - Boom-bap hip-hop (syncopated kicks, snare emphasis)
    - Four-on-floor EDM (kick every beat)
  - Add variations to each

**Independent Practice (15 minutes)**
- Exercise 3: Groove Workshop
  - Take mechanical pattern
  - Add swing (timing adjustment)
  - Vary velocity (dynamics)
  - Apply humanization
  - Compare before/after

**Closure & Assessment (5 minutes)**
- Listen to best grooves from class
- Discuss what made them feel good

#### Creative Challenge: Genre Mashup Beat

**Assignment Overview:**
Create a drum pattern that combines elements from 2 different genres.

**Instructions:**
1. Choose 2 distinct genres (e.g., rock + hip-hop, EDM + jazz, reggae + trap)
2. Research characteristic drum patterns for each
3. Create 32-bar pattern with:
   - A section: Emphasis on genre 1
   - B section: Emphasis on genre 2
   - Transition fills between sections
4. Include at least 6 different drum sounds
5. Apply swing/humanization appropriately
6. Write brief explanation of genre fusion approach

**Assessment Criteria:**
- Genre knowledge (20%): Demonstrates understanding of both genres
- Programming (40%): Accurate patterns, good transitions, proper fills
- Groove (30%): Feels good, danceable, musical
- Documentation (10%): Explains creative decisions

### Lesson 6: Layering Sounds

**Duration:** 90 minutes
**Difficulty:** Beginner-Intermediate

#### Lesson Flow

**Opening (10 minutes)**
- Play solo kick drum (weak)
- Play layered kick (3 samples: sub, mid, click) (powerful)
- Question: "What's the difference?"
- Introduction to frequency spectrum and layering strategy

**Direct Instruction (20 minutes)**
- Mix-agent explains:
  - Frequency spectrum (20Hz-20kHz)
  - Bass (20-250Hz), Mids (250Hz-4kHz), Highs (4kHz-20kHz)
  - Layering principle: Each layer occupies different frequency range
  - Phase alignment basics
  - EQ as a carving tool
- Demonstration: Building a super kick
  - Layer 1: Sub bass (40-60Hz)
  - Layer 2: Body (80-200Hz)
  - Layer 3: Click/attack (2-5kHz)

**Guided Practice (40 minutes)**
- Exercise 1: Frequency Map (20 min)
  - Students load multiple drum sounds
  - Use spectrum analyzer to visualize frequency content
  - Identify primary frequency range of each
  - Map sounds on frequency chart

- Exercise 2: Kick Layering (25 min)
  - Select 3 different kick samples
  - Layer in Zordic
  - Use EQ to emphasize different frequencies in each:
    - Sample 1: Low-pass filter, boost sub
    - Sample 2: Band-pass for body
    - Sample 3: High-pass for click
  - Balance levels
  - Compare to single kick

**Independent Practice (15 minutes)**
- Exercise 3: EQ Carving
  - Take existing layered beat
  - Use EQ on each element to create space
  - Cut conflicting frequencies
  - Boost characteristic frequencies
  - A/B compare processed vs unprocessed

**Closure & Assessment (5 minutes)**
- Class discussion: When to layer vs when to use single sound?
- Mix-agent provides pro tips

#### Creative Challenge: Super Drum Designer

**Assignment Overview:**
Design 3 custom drum sounds using layering techniques.

**Instructions:**
Create:
1. **Super Kick:**
   - 2-4 layers
   - Sub layer (below 80Hz)
   - Mid layer (80-500Hz)
   - High layer (attack/click)
   - Use EQ on each layer
   - Final combined kick should be powerful and clear

2. **Mega Snare:**
   - 2-3 layers
   - Body layer (200-400Hz)
   - Snap layer (2-5kHz)
   - Optional: noise/ambience
   - EQ for punch and clarity

3. **Hybrid Clap:**
   - 2-3 layers
   - Experiment with combining different transient sounds
   - Create unique clap sound

4. Documentation:
   - List each layer used
   - Describe EQ and processing applied
   - Explain creative decisions

**Assessment Criteria:**
- Technique (50%): Proper layering, effective EQ, phase coherence
- Sound quality (30%): Professional results, clear and punchy
- Documentation (20%): Thorough process explanation

### Lesson 7: Creating Grooves

**Duration:** 90 minutes
**Difficulty:** Intermediate

#### Lesson Flow

**Opening (10 minutes)**
- Play two beats:
  1. Mechanical, perfectly quantized
  2. Same pattern with groove, ghost notes, dynamics
- Vote: Which feels better?
- Introduction: "Groove is the feel, not just the pattern"

**Direct Instruction (20 minutes)**
- Rhythm-agent explains:
  - Groove = pocket + feel + dynamics
  - Ghost notes (low-velocity hits between main beats)
  - Accent patterns (emphasizing certain hits)
  - Syncopation (off-beat emphasis)
  - Push and pull (slight timing variations)
  - The importance of space
- Live demonstration of adding ghost notes to basic pattern

**Guided Practice (40 minutes)**
- Exercise 1: Groove Analysis (20 min)
  - Listen to classic grooves:
    - James Brown "Funky Drummer"
    - J Dilla hip-hop beat
    - Daft Punk "Around the World"
  - Identify groove elements
  - Agent helps deconstruct patterns
  - Attempt to recreate in Zordic

- Exercise 2: Ghost Note Master (25 min)
  - Start with basic pattern
  - Add ghost notes on snare (velocity 30-50)
  - Add subtle hi-hat variations
  - Experiment with placement
  - Agent provides real-time groove analysis

**Independent Practice (15 minutes)**
- Exercise 3: Fill Factory
  - Learn fill placement (end of 4 or 8 bars)
  - Create 5 different fills (2, 4, 8 beat fills)
  - Use toms, cymbals, snare rolls
  - Practice transitioning back to main groove

**Closure & Assessment (5 minutes)**
- Share best grooves
- Rhythm-agent analyzes groove quality

#### Creative Challenge: The Perfect Loop

**Assignment Overview:**
Create an 8-bar drum groove that loops seamlessly with maximum groove potential.

**Instructions:**
1. Create main groove (8 bars):
   - Interesting kick pattern (not just 1 & 3)
   - Snare with ghost notes
   - Hi-hat with subtle variations
   - Use dynamics (velocity 40-127 range)

2. Create variation (8 bars):
   - Modify main groove
   - Change hi-hat pattern or add percussion
   - Keep same foundation

3. Design transition fill (1-2 beats):
   - Smoothly connects back to main groove
   - Creates anticipation

4. Requirements:
   - Must loop perfectly
   - Must make you want to move
   - Include at least 5 ghost notes per 8 bars
   - Show dynamic range

5. Test: Loop for 2 minutes - does it stay interesting?

**Assessment Criteria:**
- Groove (40%): Feel, pocket, makes you want to move
- Technique (30%): Ghost notes, dynamics, clean execution
- Variation (20%): B section differs meaningfully
- Loop-ability (10%): Transitions smoothly, no awkward moments

### Lesson 8: Using Samplers

**Duration:** 90 minutes
**Difficulty:** Intermediate

#### Lesson Flow

**Opening (15 minutes)**
- Play famous sample-based tracks:
  - "Rapper's Delight" (Chic sample)
  - Kanye West "Through the Wire" (Chaka Khan sample)
  - Daft Punk "Harder Better Faster Stronger" (Edwin Birdsong sample)
- Discuss: "How did producers transform these samples?"
- Introduction to sampling as creative tool

**Direct Instruction (20 minutes)**
- Sample-agent demonstrates:
  - What is a sampler?
  - Recording audio into sampler
  - Sample editing (trim, normalize, fade)
  - Looping techniques
  - Time-stretching and pitch-shifting
  - Mapping samples across keyboard
  - Slice modes for rhythmic material
- Live demo: Sample to instrument in Zordic

**Guided Practice (35 minutes)**
- Exercise 1: Sample Safari (25 min)
  - Students record 5 environmental sounds:
    - Percussive (door slam, book drop)
    - Tonal (glass clink, vocal phrase)
    - Textural (rain, static)
  - Import into Zordic sampler
  - Edit and clean samples
  - Create loops

- Exercise 2: Melodic Sampler (20 min)
  - Take one interesting sound
  - Map across keyboard
  - Use pitch-shift to create playable instrument
  - Play a simple melody
  - Adjust envelope for musicality

**Independent Practice (15 minutes)**
- Exercise 3: Chop Shop
  - Load a drum loop
  - Use slice mode to chop into individual hits
  - Rearrange slices to create new pattern
  - Export new rhythm

**Closure & Assessment (5 minutes)**
- Share most creative sampled instrument
- Sample-agent provides advanced techniques preview

#### Creative Challenge: Found Sound Beat

**Assignment Overview:**
Create a complete beat using only samples you recorded yourself (no preset drums).

**Instructions:**
1. Recording phase:
   - Record at least 8 original samples
   - Include: 3 percussive, 2 tonal, 3 textural
   - Can record anything (household objects, voice, environment)

2. Processing phase:
   - Edit samples in Zordic
   - Create:
     - Kick drum from sample
     - Snare drum from sample
     - Hi-hat from sample
     - At least one melodic element from sample

3. Composition phase:
   - Create 16-bar beat
   - Use only your sampled instruments
   - Include variation and dynamics
   - Make it groove!

4. Documentation:
   - List original sounds recorded
   - Explain how each was transformed
   - Describe processing applied

**Assessment Criteria:**
- Creativity (40%): Unique sample choices, creative transformation
- Technical (35%): Clean recordings, effective processing, functional drums
- Musicality (15%): Grooves well, interesting composition
- Documentation (10%): Thorough explanation of process

## Unit 3: Melody Creation (Weeks 5-6)

### Lesson 9: Scale Selection and Key

**Duration:** 90 minutes
**Difficulty:** Intermediate

#### Lesson Flow

**Opening (10 minutes)**
- Play same melody in major, minor, dorian, phrygian
- Question: "How does the mood change?"
- Introduction: "Scales are emotional palettes"

**Direct Instruction (25 minutes)**
- Theory-agent presents:
  - Review: Major and minor scales
  - Introduction to modes:
    - Ionian (major) - bright, happy
    - Dorian - jazzy, sophisticated
    - Phrygian - Spanish, dark
    - Lydian - dreamy, ethereal
    - Mixolydian - bluesy, rock
    - Aeolian (minor) - sad, serious
    - Locrian - unstable, tense
  - Pentatonic scales (major and minor)
  - Blues scale
  - Choosing scale for desired emotion
  - Key signatures and staying in key

**Guided Practice (35 minutes)**
- Exercise 1: Scale Emotion Wheel (20 min)
  - Agent plays each mode
  - Students associate with emotions/colors/images
  - Create emotion map for scales
  - Discuss cultural associations

- Exercise 2: Modal Explorer (30 min)
  - Students experiment with each mode in Zordic
  - Use scale-highlight feature
  - Play simple melodies in each mode
  - Record impressions
  - Identify favorite mode

**Independent Practice (15 minutes)**
- Exercise 3: Key Consistency
  - Choose a key (e.g., D major)
  - Enable scale snap in piano roll
  - Compose 8-bar melody staying in key
  - Agent verifies no wrong notes

**Closure & Assessment (5 minutes)**
- Quick quiz: Play mode, students identify
- Discuss: "When might you break the scale rules?"

#### Creative Challenge: Modal Journey

**Assignment Overview:**
Create 7 short melodies, one in each mode of the major scale.

**Instructions:**
1. Use C as root for all modes:
   - C Ionian (C major scale)
   - C Dorian (C-D-Eb-F-G-A-Bb-C)
   - C Phrygian (C-Db-Eb-F-G-Ab-Bb-C)
   - C Lydian (C-D-E-F#-G-A-B-C)
   - C Mixolydian (C-D-E-F-G-A-Bb-C)
   - C Aeolian (C minor scale)
   - C Locrian (C-Db-Eb-F-Gb-Ab-Bb-C)

2. For each mode:
   - Compose 8-bar melody
   - Try to express mode's unique character
   - Use mode's distinctive notes
   - Choose appropriate synth sound

3. Label each clearly
4. Write brief description of each mode's feeling

**Assessment Criteria:**
- Theory (40%): Correct mode construction, stays in scale
- Expression (40%): Captures mode character, musical phrasing
- Organization (10%): Clear labeling, documentation
- Exploration (10%): Shows understanding of mode differences

### Lesson 10: Melodic Composition

**Duration:** 90 minutes
**Difficulty:** Intermediate

#### Lesson Flow

**Opening (10 minutes)**
- Play iconic melodic hooks:
  - "Somewhere Over the Rainbow"
  - "Super Mario Bros" theme
  - "Eye of the Tiger"
- Question: "What makes these unforgettable?"
- Introduction to melodic composition principles

**Direct Instruction (20 minutes)**
- Melody-agent explains:
  - Motif (short musical idea, 2-4 notes)
  - Development techniques:
    - Repetition (same motif)
    - Sequence (motif at different pitches)
    - Inversion (motif upside down)
    - Retrograde (motif backwards)
    - Augmentation/diminution (slower/faster)
  - Melodic contour (shape):
    - Ascending (builds energy)
    - Descending (releases tension)
    - Arch (balanced)
    - Static (creates stability)
  - Phrasing (musical sentences)
  - Call and response
  - Tension and release

**Guided Practice (40 minutes)**
- Exercise 1: Motif Development (25 min)
  - Create simple 3-note motif
  - Develop using each technique:
    - Repeat exactly
    - Sequence up/down
    - Invert
    - Play backwards
  - Combine into 16-bar melody

- Exercise 2: Contour Analysis (20 min)
  - Analyze famous melodies:
    - "Happy Birthday" - arch contour
    - "Stairway to Heaven" - ascending
  - Draw contour shapes
  - Recreate similar contours with new notes

**Independent Practice (15 minutes)**
- Exercise 3: Call and Response
  - Create 2-bar "question" phrase (ends on unstable note)
  - Create 2-bar "answer" phrase (ends on root)
  - Repeat pattern with variations
  - Build to 16 bars

**Closure & Assessment (5 minutes)**
- Share most memorable melody
- Class sings back best hooks
- Melody-agent analyzes what made them stick

#### Creative Challenge: The Hook Challenge

**Assignment Overview:**
Compose a catchy 4-bar hook that people can't forget.

**Instructions:**
1. Create memorable 4-bar melodic phrase
2. Requirements:
   - Clear motif that repeats
   - Interesting contour
   - One surprising moment (unexpected note/rhythm)
   - Singable/hummable
   - Resolves satisfyingly

3. Testing phase:
   - Play for 3 different people
   - Wait 1 hour
   - Ask them to sing/hum it back
   - If they can't, revise and repeat

4. Final submission:
   - Audio of hook
   - MIDI file
   - Documentation:
     - Describe your motif
     - Explain the surprise moment
     - Report testing results

**Assessment Criteria:**
- Memorability (40%): Can people remember and reproduce it?
- Composition (30%): Uses motif, contour, phrasing effectively
- Creativity (20%): Unique, surprising, interesting
- Testing (10%): Followed test process, documented results

## General Teaching Notes

### Classroom Management
- **Headphone protocol:** All production work done with headphones to avoid chaos
- **Sharing work:** Use classroom audio system for presentations
- **Saving work:** Enforce consistent project naming: "FirstLast_LessonNumber_ProjectName"
- **Agent etiquette:** Agents are assistants, not answer-givers

### Differentiation Strategies
- **Advanced students:** Extension challenges, mentor roles, additional genre exploration
- **Struggling students:** Additional agent scaffolding, simplified challenges, partner work
- **Learning styles:** Visual (waveforms, spectrum), auditory (listening), kinesthetic (controllers)

### Technology Troubleshooting
- **Backup plan:** Always have offline alternatives
- **Agent issues:** Agents can be disabled; have manual tutorials ready
- **Audio issues:** Test equipment before each lesson
- **File management:** Weekly project backups, cloud storage

### Assessment Best Practices
- **Formative:** Check understanding during guided practice
- **Agent analytics:** Review student-agent interactions for insights
- **Self-assessment:** Students rate their own work before submission
- **Peer review:** Structured peer feedback on projects
- **Portfolio review:** Mid-semester check-ins on progress

### Cultural Responsiveness
- **Genre diversity:** Include music from many cultures
- **Student choice:** Let students bring their own musical interests
- **Representation:** Use examples from diverse artists
- **Language:** Avoid assuming musical background

### Safety and Ethics
- **Hearing protection:** Educate about volume levels, hearing damage
- **Copyright:** Teach about sampling rights, fair use, royalty-free resources
- **Digital citizenship:** Respectful collaboration, giving credit
- **Accessibility:** Ensure all students can access technology
